Python Advanced Calculator

A beginner-friendly Python calculator project that supports:

->Arithmetic operations
->Bitwise operations
->Logical operations
->Percentage calculation
Minimum & Maximum finder
Error handling for division by zero
Features
Arithmetic Operations:
->Addition (+)
->Subtraction (-)
->Multiplication (*)
->Division (/)
->Floor Division (//)
->Modulus (%)
->Power (**)
Bitwise Operations:
->Bitwise OR (|)
->Bitwise AND (&)
->Bitwise XOR (^)
->Bitwise NOT (~)
->Left Shift (<<)
->Right Shift (>>)
Logical Operations:
->Logical OR (or)
->Logical AND (and)
Other Utilities:
->Percentage Calculation (percent)
->Minimum Number (min)
-?Maximum Number (max)
